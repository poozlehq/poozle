## Authentication Specification

We support OAuth2 based Authentication.

## Supported Models

- [Calendar](/calendar/calendar/)

# Mapping

## FreeBusy
| Key           | Value         | Description   |
| ------------- |:-------------:|:-------------:|
| `id`          | `id`          | The ID of the calendar |
| `description` | `description` | The description of the calendar |
| `timezone`    | `timezone`    | The timezone of the calendar |
| `start_time`  | `start_time`  | The start_time of the freeB |
| `end_time`    | `end_time`    | The end_time of the calendar |
| `status`      | `status`      | The status of the calendar |
